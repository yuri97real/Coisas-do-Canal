# Barra de Progresso de Upload

## Descrição

Formulário para envio de arquivos com exibição de progresso do upload em tempo real.

## Instruções

Você pode iniciar o projeto com ferramentas como o Xampp, Laragon, Wamp ou o próprio servidor embutido do PHP:

    php -S localhost:3333

## Observações

O código não conta com validações, leitura de arquivos e etc. Ele está na sua forma mais pura. No entanto, as regras de negócio que você irá implementar, não irão influenciar no seu funcionamento.

O objetivo do projeto é apenas obter dados do upload durante seu progresso.
